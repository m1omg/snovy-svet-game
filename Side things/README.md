# Side things 🧪

Vedľajší priestor (scratchpad) na "vibecoding" – experimenty a malé projekty mimo hlavnej hry
**Snový Svet**. Hlavná hra (`index.html`, `snovy-3d/`) sa tohto priečinka nedotýka.

## Načo to je
- Skúšať nové nápady bez zásahu do hry.
- Mať to v tej istej Claude Code session (netreba druhú inštanciu).

## Ako to používame
- Každý mini-projekt nech má vlastný podpriečinok, napr. `Side things/<nazov-projektu>/`.
- Hlavná hra a tento priečinok sú nezávislé.

## Stav
- Zatiaľ prázdne – čaká sa, čo ideme robiť. 🙂
